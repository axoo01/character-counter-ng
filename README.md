# Character Counter App

A real-time text analysis tool built with **Angular 18**. This application helps users analyze their written content with precision, providing instant metrics on character counts, word counts, sentence counts, and letter density.

Originally developed as a vanilla JavaScript lab, this version has been completely re-architected into a modular Angular application to demonstrate best practices in component-based development.

[Live Demo](https://your-app-name.netlify.app/)

## Features

- **Real-Time Analysis**: Instant updates for character, word, and sentence counts as you type.
- **Space Exclusion Toggle**: Option to calculate character counts with or without spaces.
- **Character Limit & Warnings**: Set a custom character limit and receive immediate visual feedback and warning messages when the limit is exceeded.
- **Reading Time Estimator**: Provides an approximate reading time based on standard reading speeds.
- **Letter Density Breakdown**: A visual percentage-based breakdown of alphabetic character usage with "See More" functionality.
- **Dynamic Theming**: Seamlessly switch between light and dark modes with persistent state management and smooth icon animations.
- **Pixel-Perfect Responsive Design**: A fully adaptive layout that scales gracefully from mobile to ultra-wide desktops.
- **Accessible UI**: Keyboard navigation support and clear focus indicators for a better user experience.
